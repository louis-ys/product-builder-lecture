# Project Blueprint: Product Builder - Theme Support

## Overview
This project is a simple web application designed to demonstrate modern web standards and responsive design. It includes a theme-switching feature (Light/Dark mode) that persists user preferences using `localStorage`.

## Project Structure & Features
- **Modern HTML/CSS/JS:** Framework-less approach using standard web features.
- **Theme Support:** Dynamic switching between Light and Dark modes.
- **Persistence:** Remembers the user's theme choice across sessions.
- **Web Components:** Encapsulated UI elements for better maintainability.

## Implementation Plan (Current Task)
1. **Define Color Tokens:** Use CSS custom properties in `style.css`.
2. **Implement Theme Logic:** Add theme-switching functionality in `main.js`.
3. **Add Toggle UI:** Create a theme toggle button in `index.html`.
4. **Git Deployment:** Commit and push the changes to GitHub.
