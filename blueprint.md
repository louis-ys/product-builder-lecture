# Project Blueprint: Product Builder - Theme Support & Contact Form

## Overview
This project is a simple web application designed to demonstrate modern web standards and responsive design. It includes a theme-switching feature (Light/Dark mode) and a partnership inquiry form integrated with Formspree.

## Project Structure & Features
- **Modern HTML/CSS/JS:** Framework-less approach using standard web features.
- **Theme Support:** Dynamic switching between Light and Dark modes with persistent storage.
- **Partnership Inquiry Form:** A clean, responsive form connected to Formspree for handling inquiries.
- **Visual Design:** Polished UI with card layouts, responsive spacing, and theme-aware styling for all components.

## Implementation Plan (Current Task)
1. **Define Color Tokens:** Use CSS custom properties in `style.css` for both light and dark themes.
2. **Implement Theme Logic:** Persistent theme switching in `main.js`.
3. **Add Partnership Form:** Integrated Formspree inquiry form in `index.html`.
4. **Style Enhancements:** Added card layouts and improved typography in `style.css`.
5. **Git Deployment:** Commit and push changes to GitHub.
